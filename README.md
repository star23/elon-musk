> "I am obsessed with truth. Pathologically." — Elon Musk

Ask any question — technology, startups, politics, philosophy, the future of humanity — and get an answer powered by Elon Musk's thinking frameworks.

This skill distills Elon's core mental models from [*The Book of Elon*](https://www.elonmuskbook.org/) by [Eric Jorgenson](https://x.com/EricJorgenson), and applies them to YOUR questions. First-principles thinking, The Algorithm, thinking in limits, the Idiot Index — all at your fingertips.

**This is not role-play.** Claude doesn't pretend to be Elon. It applies his publicly documented thinking frameworks to analyze problems the way he would — then gives you an actionable answer.

## What's Inside

**5 Core Thinking Tools:**

| Tool | What It Does |
|------|-------------|
| First-Principles Thinking | Strip away assumptions, reason from fundamental truths |
| The Algorithm | 5-step optimization: Question → Delete → Simplify → Accelerate → Automate |
| Thinking in Limits | Push problems to extremes to reveal hidden structure |
| The Magic-Wand Number | Define the theoretically perfect state, then close the gap |
| The Idiot Index | Finished price / Raw material price = how much waste exists |

**Plus:** 69 core Musk methods, key real-world examples, and a response style guide that captures his directness, quantitative thinking, and contrarian instincts.

## How It Thinks

Every response follows a flexible framework:

1. **Challenge the question itself** — Are the assumptions valid? Are we asking the wrong question?
2. **First-principles breakdown** — What are the fundamental facts? What do physics allow?
3. **Limits test** — What happens at extremes? What's the theoretically perfect state?
4. **Apply The Algorithm** — Question → Delete → Simplify → Accelerate → Automate
5. **Action plan** — What should be done? What's the bottleneck? Where's the urgency?
6. **Honest about risks** — Where might this be wrong? What signals would change the conclusion?

## 思考方式

每次回答都遵循灵活的框架：

1. **质疑问题本身** — 假设是否成立？我们是否在问一个错误的问题？
2. **第一性原理拆解** — 最基本的事实是什么？物理定律允许什么？
3. **极限测试** — 推到极端会怎样？理论上的完美状态是什么？
4. **应用 The Algorithm** — 质疑 → 删除 → 简化 → 加速 → 自动化
5. **行动计划** — 该做什么？瓶颈在哪？紧迫性在哪？
6. **诚实面对风险** — 哪里可能错？什么信号会改变判断？
